## 2. 已锁定供应链边界

当前 Capsule Preview 只能按以下边界外放：

| 维度 | 当前允许表达 | 当前禁止表达 |
| --- | --- | --- |
| 商品身份 | 紫铜水瓶手冲咖啡壶、非银内壁版本、单一 SKU 资料预览 | 银内壁、镀银内壁、999 银内壁、多 SKU 或版本矩阵 |
| 阶段 | 资料预览、到货通知、样品待验收 | 开售、预售、预约、候补、排队、锁定名额 |
| 价格 | 价格待样品验收后公布 | 公开售价、折扣、包邮、内部测算价格带 |
| 库存 | 不锁定库存 | 现货、限量、库存数、即将售罄 |
| 交期 / 物流 | 不承诺发货时间 | 即发、预计到货日、跨境时效、美国仓承诺 |
| 检测 / 合规 | 当前未锁定第三方检测资料 | 食品级、食品安全、检测通过、重金属安全 |
| 材质 / 规格 | 紫铜壶体、非银内壁版本；图册规格必须标注待样品复核 | 把图册规格写成实测值，或承诺材质纯度 / 性能 |

## 3. 内部履约判断底线

以下内容只能作为内部判断，不能写入页面、广告、客服承诺或外部渠道：

- `$119` 当前不成立，不能作为对外价格锚点。
- `$149` 只在 `中国直发美国` 路线下勉强进入最小可行带。
- `$179` 才开始让 `美国仓 / 3PL` 有空间，但仍需要真实仓配报价验证。
- 当前没有批量价、MOQ、真实分区物流报价、HS 编码、税费口径、3PL 报价、破损率和退款率数据。
- 当前推荐只验证 `资料预览 + 到货通知` 兴趣信号，不验证交易。

## 4. D7 风险扫描口径

D7 只回答一个问题：首轮外放后，是否出现用户或渠道对供应链边界的集中追问或误读。

### 4.1 必查来源

- Shopify 到货通知表单备注或关联标签。
- 客服 / 私域 / 邮件回复中关于产品的直接问题。
- 外放渠道文案、落地页、PDP、首页和 FAQ 的实际线上版本。
- 任何内部或外部转发截图中出现的价格、库存、交期、物流、检测或材质表述。

### 4.2 必扫问题类型

| 类别 | 风险问题示例 | D7 判断口径 |
| --- | --- | --- |
| 价格 | “多少钱”“是否包邮”“能不能预订这个价” | 页面不得给价格；如用户集中追问价格，只记录需求，不补承诺 |
| 库存 | “有现货吗”“是不是限量” | 必须保持不锁定库存；任何限量/现货暗示都需修正文案 |
| 交期 | “什么时候发货”“多久到美国” | 当前不能答具体时效；如渠道暗示即发，需同日升级 |
| 物流 | “美国仓发吗”“运费多少” | 美国仓和直发只能是内部路线假设，不能对外承诺 |
| 检测 | “是否食品安全”“有没有 FDA/LFGB” | 当前不能宣称检测或食品接触安全结论 |
| 材质 | “是不是银内壁”“是不是纯铜” | 必须坚持非银内壁版本；规格和材质待复核项不能写死 |

## 5. D14 累计复盘口径

D14 不只看有没有单条问题，而是判断外部预期是否已经与 `notify-only / sample_verified=false` 边界冲突。

### 5.1 累计判断维度

| 风险等级 | 判断标准 | 建议动作 |
| --- | --- | --- |
| 低 | 问题分散，用户能接受资料预览与到货通知边界；页面无越界承诺 | 继续观察，维持入口 |
| 中 | 多次集中追问价格、交期、检测或材质，说明当前页面解释不够；但尚无实际承诺越界 | 修正文案，补充 FAQ 或透明说明，必要时收窄外放渠道 |
| 高 | 外部已形成“可购买 / 有价格 / 有库存 / 有交期 / 有检测背书 / 银内壁”的误读，或内部人员把占位测算对外引用 | 同日升级，建议收窄入口并修正文案 |
| Rollback 临界 | 购买、预售、预约、价格、库存、发货、食品安全、检测通过等承诺已被公开传播，或购买入口误开 | 建议立即 rollback 或关闭入口，保留证据并回滚文案 / 配置 |

## 6. 同日升级条件

发现以下任一情况，不等 D7 / D14，必须同日升级并指出来源：

- 对外页面、FAQ、客服话术、邮件或渠道文案出现正式售价、折扣、包邮阈值或 `$119 / $149 / $179` 内部测算。
- 出现现货、限量、即发、预计到货日、美国仓发货、跨境时效或运费承诺。
- 出现食品级、食品安全、FDA、LFGB、检测通过、重金属安全等合规背书。
- 非银内壁版本被写成银内壁、镀银内壁、999 银内壁或 `silver-lined`。
- `sample_verified=false` 时出现加购、结账、预约、候补、预售、锁定库存或购买型按钮。
- 图册规格、材质字段或内部履约占位值被写成已实测、已确认或已承诺。
