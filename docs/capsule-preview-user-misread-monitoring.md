## 4. 必查入口

外放后，优先检查以下入口的真实线上版本与用户反馈：

- 首页、PDP、FAQ、页头提示条、到货通知表单确认文案。
- Shopify 到货通知表单备注、标签和导出记录。
- 客服、私域、邮件回复和任何对外 DM 中与该商品相关的对话。
- 渠道投放文案、社媒贴文、短链落地页、站外转发截图。
- 内部成员转发给外部对象的图片、话术或摘要，如出现对外扩散也应计入证据。

## 5. 分类口径

### 5.1 价格误读

观察问题：

- 用户是否把资料预览页理解成“故意不写价格的销售页”。
- 是否出现“先告诉我价格我就下单”“现在付定金能锁价吗”等询问。

越线表达：

- 任何正式售价、折扣、包邮门槛、定金金额。
- 任何 `$119 / $149 / $179` 或其他内部测算价对外出现。

### 5.2 履约误读

观察问题：

- 用户是否把 `notify-only` 误读成“马上可发货”或“已有库存待卖”。
- 是否集中追问现货、发货日、美国仓、运费、到货时效。

越线表达：

- 现货、限量、即发、预计到货时间、美国仓发货、跨境时效、运费承诺。
- 任何“已经可以排单”“马上补货”“本周可发”的说法。

### 5.3 性能误读

观察问题：

- 用户是否把图册、视觉表达或 FAQ 误读成已验证的性能结论。
- 是否出现“是否适合明火”“控流稳不稳”“保温多久”“是否食品级”这类集中追问。

越线表达：

- 未经验证的控流、导热、保温、耐用、抗氧化、食品安全、认证背书。
- 把图册描述、样品待验收字段或内部判断写成已实测性能。

### 5.4 预售 / 候补误读

观察问题：

- 用户是否把到货通知表单理解成排队、预约或锁定购买资格。
- 是否出现“我先登记是不是就能优先买到”“现在能不能占位”等询问。

越线表达：

- 预售、预约、候补、排队、锁库存、锁名额、优先购买资格。
- 任何暗示“先留邮箱就等于进入购买名单”的表达。

## 8. 同日升级条件

外放后发现以下任一情况，不等 D7 / D14，必须当日升级并保留来源证据：

- 页面、FAQ、客服或渠道文案出现正式售价、折扣、定金、包邮或内部测算价格。
- 出现现货、限量、发货日、时效、运费、美国仓、交期承诺。
- 出现控流、导热、保温、食品安全、检测通过、认证背书或其他未验证性能结论。
- 出现预售、预约、候补、排队、优先购买、锁库存或锁名额表达。
- `sample_verified=false` 时出现加购、结账、支付、预约或其他购买型交互。
