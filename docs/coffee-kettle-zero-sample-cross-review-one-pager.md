## Cross-review 结论

| 检查项 | 结果 | 说明 |
| --- | --- | --- |
| `非银内壁版本` 红线 | 通过 | docs 与 theme 当前都以 `非银内壁版本` 为唯一对外口径；未发现仍把该款写成银内壁款的前台模板字段 |
| `单一 SKU` 红线 | 通过 | 当前资料、schema 和模板可按单 SKU 推进；默认 variant 选择器风险已在模板侧压掉，不再把默认变体渲染成可选版本 |
| `图册规格待复核` 红线 | 通过 | 容量与尺寸保留为图册占位值，并在 schema / 页面说明中保留 `catalog_pending_sample` 或 `待样品复核` 语义 |
