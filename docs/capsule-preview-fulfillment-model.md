# Capsule Preview 最小履约测算框架

> 对应任务：[HTH-234](/HTH/issues/HTH-234)  
> 适用阶段：Capsule Preview / 资料预览 + 到货通知  
> 结论性质：内部保守占位测算，不构成最终财务结论  
> 结论日期：2026-04-20

## 1. 目的与边界

这份测算只回答一个问题：在 `不下单、不送样、不补新询盘` 的前提下，当前这把 `非银内壁紫铜水瓶手冲咖啡壶` 是否已经具备最小的履约与价格空间判断，足以支持 CEO 做 `go / no-go`。

本框架只比较两条路线：

- `中国直发美国`
- `美国仓 / 3PL`

本框架只给三档售价假设：

- `$119`
- `$149`
- `$179`

这不是最终定价建议，只是给当前预览态提供一个最小判断面板。

## 2. 当前可用输入

当前只使用仓库中已经锁定、且允许内部引用的输入：

- 样品价占位：约 `¥260`

- 毛重占位：约 `1 kg`
- 当前商品阶段：`draft / notify-only / sample_verified=false`

## 3. 占位假设

由于当前没有批量报价、真实物流报价、3PL 报价和关税归类结论，以下都按保守占位处理：

| 项目 | 占位值 | 说明 |
| --- | --- | --- |
| 汇率占位 | `1 USD = 7.3 CNY` | 仅用于内部换算，不代表当日真实汇率 |
| 工厂成本占位 | `$35.6` | 用 `¥260 / 7.3`，直接拿样品价代替出厂价，故意偏保守 |
| 包装加固占位 | `$3.0` | 视为单件跨境可发的额外保护成本 |
| 中国侧交接/处理 | `$2.4` | 包含出库、贴单、集运前处理等保守占位 |
| 支付手续费 | `3.6% + $0.30` | 用于 Shopify 独立站的最小支付损耗占位 |
| 损耗/破损准备金 | `$4.0 / 单` | 两条路线都保留一档基础售后缓冲 |
| 目标贡献毛利率 | `45%` | 目的是留出 CAC、内容制作、退款和管理误差空间 |

说明：

- 当前没有拿到批量 MOQ 与阶梯价，所以 `工厂成本` 故意不按大货价算，避免把利润看得过好。
- 当前没有拿到真实 USPS/UPS/跨境专线或 3PL 报价，所以 `路线成本` 只做最小方向判断。
- `break-even CAC` 这里按最简公式处理，不含团队固定成本、软件订阅和税务尾差。

## 4. 两条物流路线的 landed cost 框架

### 4.1 中国直发美国

公式：

`landed cost = 工厂成本占位 + 包装加固 + 中国侧交接 + 跨境包裹末端派送 + 损耗准备金`

占位拆分：

| 成本项 | 占位值 |
| --- | --- |
| 工厂成本占位 | `$35.6` |
| 包装加固 | `$3.0` |
| 中国侧交接/处理 | `$2.4` |
| 跨境包裹 + 美国末端派送 | `$28.0` |
| 损耗/破损准备金 | `$4.0` |
| `直发 landed cost` | `$73.0` |

判断：

- 优点：不压美国库存，不预付 3PL 固定处理费，适合 `预览态 / 小单量验证`。
- 缺点：单件包裹成本高，时效与破损不确定性更高，不适合提前做履约承诺。

### 4.2 美国仓 / 3PL

公式：

`landed cost = 工厂成本占位 + 包装加固 + 中国侧交接 + 头程/清关分摊 + 3PL 入仓仓储分摊 + 拣货打包 + 美国境内派送 + 损耗准备金`

占位拆分：

| 成本项 | 占位值 |
| --- | --- |
| 工厂成本占位 | `$35.6` |
| 包装加固 | `$3.0` |
| 中国侧交接/处理 | `$2.4` |
| 头程/清关分摊 | `$14.0` |
| 入仓/仓储分摊 | `$4.0` |
| 拣货打包 | `$5.0` |
| 美国境内派送 | `$13.0` |
| 损耗/破损准备金 | `$4.0` |
| `美国仓 / 3PL landed cost` | `$81.0` |

判断：

- 优点：用户侧履约体验更容易做稳定，后续若转向正式售卖更好扩。
- 缺点：在单 SKU、低单量、未验证需求的阶段，固定处理费更重，价格压力更大。

## 5. 统一计算公式

### 5.1 含物流贡献毛利

`贡献毛利 = 售价 - 支付手续费 - landed cost`

其中：

`支付手续费 = 售价 x 3.6% + $0.30`

### 5.2 Break-even CAC

`break-even CAC = 贡献毛利`

这里的意思是：如果只看单笔订单，不计固定开销，单笔最多可承受多少 CAC 仍不亏单。

### 5.3 包邮阈值

目标设为 `贡献毛利率 >= 45%` 时：

`包邮阈值 = (landed cost + $0.30) / (1 - 3.6% - 45%)`

测算结果：

- 中国直发美国：约 `$143`
- 美国仓 / 3PL：约 `$158`

## 6. 三档售价假设结果

### 6.1 中国直发美国

| 售价 | 支付手续费 | landed cost | 贡献毛利 | 贡献毛利率 | break-even CAC | 包邮判断 |
| --- | --- | --- | --- | --- | --- | --- |
| `$119` | `$4.58` | `$73.0` | `$41.42` | `34.8%` | `$41.42` | 不建议包邮 |
| `$149` | `$5.66` | `$73.0` | `$70.34` | `47.2%` | `$70.34` | 可做最低可行包邮 |
| `$179` | `$6.74` | `$73.0` | `$99.26` | `55.5%` | `$99.26` | 包邮空间相对充足 |

结论：

- `$119` 对中国直发也偏紧，不适合做含运费的 paid acquisition。
- `$149` 是当前直发路线的最小可行带，能给预览态留下基本操作空间。
- `$179` 才是更稳的直发带，但前提是竞品样本与内容叙事能撑住更高价格锚点。

### 6.2 美国仓 / 3PL

| 售价 | 支付手续费 | landed cost | 贡献毛利 | 贡献毛利率 | break-even CAC | 包邮判断 |
| --- | --- | --- | --- | --- | --- | --- |
| `$119` | `$4.58` | `$81.0` | `$33.42` | `28.1%` | `$33.42` | 不建议包邮 |
| `$149` | `$5.66` | `$81.0` | `$62.34` | `41.8%` | `$62.34` | 低于目标阈值 |
| `$179` | `$6.74` | `$81.0` | `$91.26` | `51.0%` | `$91.26` | 进入可包邮区间 |

结论：

- `$119` 在美国仓路线下几乎没有可用空间。
- `$149` 若走美国仓，仍低于这份框架设定的 `45%` 目标贡献毛利率。
- `$179` 才开始让美国仓 / 3PL 有成立空间，但仍需要真实仓配报价验证。

## 7. Go / No-Go 倾向

### 7.1 对三档售价的判断

| 售价带 | 判断 | 原因 |
| --- | --- | --- |
| `$119` | `No-Go` | 两条路线都太紧，包邮和付费获客空间都不足 |
| `$149` | `Conditional Go` | 只对 `中国直发美国` 勉强成立；更适合预览态、自然流量、通知收集 |
| `$179` | `Conditional Go` | 两条路线都开始成立，但需要竞品样本确认价格带是否站得住 |

### 7.2 路线推荐

当前更建议先测 `中国直发美国`，不建议先上 `美国仓 / 3PL`。

原因只有三点：

- 当前阶段是 `预览态 + 到货通知`，不是高确定性发货阶段，没必要先背 3PL 固定成本。
- 当前只有单 SKU，且需求强度未验证，美国仓路线对订单密度要求更高。
- 即使售价来到 `$149`，中国直发还有最小成立空间；美国仓在同价位仍偏紧。

## 8. 当前最小结论

如果 CEO 现在只需要一个最小判断，我的结论是：

`Conditional Go`，但前提是把这件事定义成 `预览态验证`，而不是正式销售。

更具体地说：

- 可以继续推进 `资料预览 + 到货通知`。
- 不建议把 `$119` 当成有效售价带。
- 如果后续竞品样本支持中档表达，`$149 + 中国直发美国` 是当前最小可行起点。
- `美国仓 / 3PL` 不应作为第一路线，除非后续确认：
  - 真实仓配报价明显优于当前占位；
  - 或者售价带能稳定站到 `$179`；
  - 或者通过套装/加价购把 AOV 拉高。

## 9. 仍缺失、必须后补的输入

以下输入缺任何一项，都意味着这份测算只能停留在 `方向判断`，不能进入最终财务结论：

- 批量出厂价、MOQ、阶梯价
- 实际发货美国的分区物流报价
- 真实包材方案下的最终体积重
- HS 编码、关税/税费处理口径
- 美国仓 / 3PL 的入仓、仓储、拣货、派送、退件报价
- 破损率、退款率、拒收率假设
- [HTH-235](/HTH/issues/HTH-235) 回传后的竞品价格带验证

## 10. CEO 可直接引用的短结论

当前这把非银内壁紫铜咖啡壶可以继续进入 `Capsule Preview`，但只能按保守占位值做最小履约判断，不能当成正式财务结论。按现有样品价、1kg 毛重和 23cm 级包装估算，`$119` 基本不成立；`$149` 只对 `中国直发美国` 勉强成立；`$179` 才开始让 `美国仓 / 3PL` 有空间。所以如果继续，我建议先按 `中国直发美国 + 中档售价假设` 做预览态验证，把美国仓留到真实需求和仓配报价都更明确之后再评估。
