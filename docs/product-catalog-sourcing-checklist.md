# 图册/PDF 选品与供应商核验清单

## 执行口径

当前阶段不单独等待市场负责人到位。图册/PDF 资料提报规范、首轮 SKU 判断标准、1688 供应商核验口径统一由运营牵头，后续市场素材、商品页、广告测试和供应链打样均以本清单作为前置输入标准。

## 当前判断边界

当前仅收到 3 张图册封面，能确认资料方向为春喜铜的手工铜茶器、手工铜咖器、手工铜餐具三个系列，封面具备东方手作、器物陈列、铜色质感等视觉方向。

但封面不足以判断是否适合 Shopify 店铺启动。首轮选品必须等待内页/规格页/报价页后才能判断以下事项：具体 SKU 数量与款式、材质是否为纯铜/黄铜/紫铜/镀铜、是否食品接触安全、容量尺寸、重量、包装、价格带、MOQ、交期、定制能力、稳定供货能力、跨境运输成本、售后风险。

## 首轮 SKU 判断的最小资料包

用户不需要一次上传完整大 PDF。每个图册系列先提供以下页面即可完成首轮筛选。

1. 目录页或 SKU 总览页
2. 每个候选 SKU 的产品主图页，优先包含正面、侧面、俯视、使用场景
3. 规格参数页，必须包含材质、尺寸、容量、重量、颜色/表面处理、适用场景
4. 包装页，必须包含单品包装方式、外箱尺寸、装箱数、毛重、净重、防摔保护方案
5. 报价页或报价截图，必须包含 MOQ、阶梯价、是否含税、是否含包装、是否支持一件代发/小批量试单
6. 工艺说明页，必须说明手工锤纹、做旧、镀层、内胆、焊接、抛光、氧化处理等工艺
7. 认证/检测页，食品接触类优先提供 LFGB、FDA 或第三方食品接触材料检测报告；至少要提供材质说明和重金属迁移/铅镉风险说明
8. 供应商基础页或 1688 店铺页截图，包含公司名称、经营年限、产地、工厂/贸易商身份、店铺等级、交易记录、服务保障
9. 现货与交期页，必须区分现货、常规生产、定制打样、大货生产周期
10. 售后与瑕疵标准页，必须说明手工产品色差、氧化、划痕、漏水、变形、包装破损的判责与补发规则

首轮优先挑 10-20 个候选 SKU，不建议一开始审完整本图册。茶器、咖器、餐具各选 3-7 个有代表性的款式即可。

## 候选 SKU 必填字段

每个候选 SKU 必须按同一张表提交，避免口头描述导致后续报价和上架信息断层。

| 字段 | 必填内容 | 运营判断用途 |
| --- | --- | --- |
| SKU 编号/产品名称 | 图册编号、1688 商品链接、中文名、英文暂定名 | 建立选品池和后续上架映射 |
| 系列分类 | 茶器、咖器、餐具、摆件/配件 | 判断店铺启动品类结构 |
| 产品图片 | 白底图、场景图、细节图、包装图 | 判断视觉卖点和上架素材可用性 |
| 材质 | 纯铜、紫铜、黄铜、镀铜、不锈钢内胆、木柄、陶瓷配件等 | 判断食品接触安全、成本、卖点真实性 |
| 食品接触部位材质 | 内壁、杯口、壶嘴、滤网、餐具接触面分别说明 | 判断是否可作为茶具/咖具/餐具销售 |
| 工艺 | 手工锤纹、做旧、鎏金/镀层、氧化、焊接、内衬、抛光 | 判断差异化与良品率风险 |
| 尺寸 | 长宽高、口径、底径 | 判断页面信息完整度和物流体积 |
| 容量/重量 | ml、g/kg，单品净重 | 判断使用场景、定价、运费 |
| 颜色/表面处理 | 原铜、古铜、黑铜、红铜、做旧程度、是否会氧化 | 判断图片一致性和售后预期 |
| 包装 | 单品包装、礼盒、外箱、防震材料、是否可贴标 | 判断跨境破损率与品牌化可能 |
| MOQ | 现货 MOQ、定制 MOQ、混批规则 | 判断启动资金压力 |
| 报价区间 | 样品价、阶梯价、含税/不含税、是否含包装 | 判断毛利和售价带 |
| 交期 | 样品、现货、小批量、大货、旺季延迟 | 判断上新节奏和库存计划 |
| 发货地 | 省市、是否可发货代仓、是否支持国内退换 | 判断头程物流与验货安排 |
| 定制能力 | Logo、包装、颜色、尺寸、套装组合、说明书 | 判断品牌化与差异化空间 |
| 认证/质检 | 食品接触检测、材质报告、重金属检测、出厂质检标准 | 判断欧美市场准入与广告风险 |
| 运费估算 | 单件体积重、装箱数、发货到货代费用、空运/海运建议 | 判断 landed cost |
| 售后条款 | 破损补发、漏水、氧化、色差、手工瑕疵标准 | 判断用户投诉和退款风险 |
| 供应状态 | 现货、排产、停产、季节性、是否稳定复购 | 判断是否适合作为启动 SKU |
| 竞品参考 | Amazon/Etsy/Shopify 类似款价格或链接 | 判断海外需求与价格锚点 |

## 用户分批提交 PDF 的建议顺序

如果 PDF 过大，建议按以下顺序分批发，不需要先发全本。

第一批：每个图册的目录页、SKU 总览页、价格页、规格页。目标是快速判断哪些系列和款式值得进入候选池。

第二批：每个系列优先 3-7 个候选 SKU 的完整详情页。优先顺序为咖器、茶器、餐具。咖器更容易做单品差异化和礼品化，茶器适合套装和文化叙事，餐具涉及食品接触与洗护售后，需更谨慎。

第三批：包装、装箱、重量、物流资料。目标是计算跨境 landed cost，剔除体积重高、易破损、包装弱的 SKU。

第四批：认证、检测、材质、供应商资质资料。食品接触类必须补齐，否则不能作为首批主推品。

第五批：定制、样品、交期、售后条款。目标是决定是否进入样品采购和首单试单。

每批资料中，优先看价格带、材质真实性、食品接触安全、包装抗损、MOQ、交期、是否有可用图片。没有这些信息，即使产品好看也不能进入启动 SKU。

## 1688 询盘最小供应商核验清单

询盘前先做页面核验。

| 核验项 | 合格标准 |
| --- | --- |
| 店铺基础 | 店铺名称、公司主体、营业执照、经营年限清晰可查 |
| 店铺认证 | 优先选择有诚信通、实力商家、工厂/产地认证、买家保障或安心购等服务标识的店铺 |
| 商品一致性 | 图册款式与 1688 链接一致，图片不是明显盗图或混杂多个风格 |
| 交易活跃度 | 近期交易、评价、回头客、发货时效记录正常 |
| 评价风险 | 重点看差评中的漏水、掉色、氧化、包装破损、尺寸不符、发货慢 |
| 价格可信度 | 同款价格与市场区间接近，过低报价需追问材质、厚度、重量 |
| 类目专注度 | 优先铜器/茶器/咖器长期经营者，不优先杂货铺 |

询盘时必须拿到明确回复。

| 询盘问题 | 必须拿到的答案 |
| --- | --- |
| 材质 | 是否纯铜/紫铜/黄铜/镀铜，食品接触部位是否有内衬或涂层 |
| 工艺 | 手工还是机器，锤纹/做旧/镀层是否批次稳定 |
| 样品 | 样品价、样品交期、样品是否可退抵大货 |
| MOQ | 现货 MOQ、混批 MOQ、定制 MOQ |
| 报价 | 10/50/100/300 件阶梯价，是否含税、包装、定制 |
| 包装 | 单品包装尺寸重量、外箱装箱数、是否支持加固包装 |
| 合规 | 是否有食品接触检测、材质报告、重金属检测；报告是否可发原件/扫描件 |
| 外贸配合 | 是否可贴英文标签、加英文说明书、贴 FNSKU/条码、发货代仓 |
| 售后 | 漏水、破损、掉色、氧化、色差、手工痕迹的补发/退款规则 |
| 交期 | 现货、补货、大货、定制分别需要几天 |
| 发货 | 发货地、国内物流费用、是否可发指定货代 |

样品前必须完成二次核验。

1. 要求供应商发送实拍视频，包含上手展示、容量测试、壶嘴/杯口/焊点/内壁/底部/包装。
2. 要求称重和尺量视频，确认规格页数据真实。
3. 要求提供最近一次出货包装照片，判断抗摔能力。
4. 要求提供检测报告或材质证明，尤其是餐具、杯具、壶具等食品接触产品。
5. 要求确认是否支持小批量补货，避免首批卖出后断货。
6. 首单建议第三方验货或到货代仓抽检，重点检查漏水、异味、划伤、氧化、变形、包装破损。

## 启动 SKU 初筛标准

首批 Shopify 启动 SKU 应优先满足以下条件。

| 维度 | 建议门槛 |
| --- | --- |
| 视觉差异化 | 海外市场一眼能看出手工铜器质感，图片可支撑高客单价 |
| SKU 复杂度 | 单品或小套装优先，避免超多配件和复杂售后 |
| 合规风险 | 食品接触资料可追溯，不清楚内壁材质或检测缺失的先不主推 |
| 破损风险 | 包装方案明确，杯口、壶嘴、把手、盖钮等脆弱点可保护 |
| 毛利空间 | 报价、包装、头程、平台费用、广告费用后仍有定价空间 |
| MOQ 压力 | 支持小批量试单或混批，首批不压过多库存 |
| 供应稳定性 | 可复购、交期稳定、可接受抽检和补发机制 |
| 内容素材 | 有高清图、工艺故事、使用场景，可支持独立站页面和广告素材 |

建议首批不要把“好看但资料不全”的款式作为主推，只放入观察池。真正进入样品池的 SKU 必须同时具备图片、规格、报价、包装、MOQ、交期、材质/检测资料。

## CEO 可直接转发给用户的中文答复

我们目前已经看到 3 个图册封面，能确认方向是手工铜茶器、咖器和餐具系列。封面视觉方向有质感，但仅凭封面还不能判断是否适合独立站启动，因为关键的 SKU、材质、规格、价格、包装、MOQ、交期和食品接触合规信息都需要看内页或供应商资料。

PDF 如果太大，可以不用一次发完整本。建议先按这个顺序分批发：

第一批先发每本图册的目录页、SKU 总览页、规格页和报价页。我们先快速筛出值得看的款式。

第二批请从茶器、咖器、餐具里各挑 3-7 个你们最想推的款式，发对应详情页。每个款式最好包含产品图、尺寸、容量、重量、材质、工艺、颜色、包装、MOQ、报价区间、交期、发货地、是否支持定制。

第三批请补包装和物流信息，包括单品包装、外箱尺寸、装箱数、毛重净重、防摔方案、国内发货地和到货代仓的大概运费。

第四批请补供应商和质检资料，尤其是 1688 店铺链接、公司主体、是否工厂、经营年限、交易评价、售后条款，以及食品接触类产品的材质证明、食品接触检测或重金属检测资料。茶具、咖具、餐具如果接触饮品或食物，这部分资料必须确认后才能作为首批主推。

如果是 1688 供应商，请同时发店铺首页截图、营业执照/认证截图、商品链接、交易和评价截图、报价聊天记录。询盘时重点问清楚：材质是否纯铜/紫铜/黄铜/镀铜，食品接触部位是什么材质，是否有检测报告，MOQ 和阶梯价，样品价和交期，包装尺寸重量，是否支持英文标签/说明书/贴标，破损、漏水、掉色、氧化和色差怎么售后。

我们收到第一批资料后，会先从运营角度判断哪些款式适合进入候选池；收到报价、包装、合规和供应商资料后，再决定哪些 SKU 进入样品采购和首批上架测试。
