# Capsule Preview 路线 CEO Go / No-Go 报告

> 对应任务：[HTH-225](/HTH/issues/HTH-225)
> 汇报日期：2026-04-20
> 结论：`Conditional Go`

## 一句话结论

当前这条 `Capsule Preview / Pre-launch Foundation` 路线值得继续，但只能按 `资料预览 + 到货通知` 的低承诺验证推进，不能当成正式开卖项目。

## 为什么继续

- 内容结构已经成立：`1 个核心商品锚点 + 2 个内容锚点 + 2 个信任锚点` 足以支撑首页与 notify-only PDP，不再只是“空店 + 单商品壳”。
- 市场带初步成立：公开样本显示，中档手工 / 设计感铜色咖啡器具存在可比参考带，本 SKU 有继续做 preview 的理由。
- 技术基线已就绪：Wokiee 主题、notify-only gating、draft 商品与 6 项最小配置已经明确，本周可以推进到可预览外放状态。

## 为什么不是完全 Go

- 按当前保守履约占位测算，`$119` 基本不成立。
- `中国直发美国 + $149` 才是最小可行组合；`美国仓 / 3PL` 当前仍然偏早。
- 市场样本支持中档参考带，但不足以支撑在预览态阶段直接贴近高端品牌价格带。

## 当前推荐路径

### 1. 先完成预览态外放，不公开价格承诺

- 完成技术侧 6 项最小配置
- 上线首页结构、notify-only PDP、到货通知表单
- 继续保持 `sample_verified=false`

### 2. 用预览态验证兴趣，而不是验证交易

- 不开购买入口
- 不做预售、预约、候补
- 不做付费投放
- 先通过私域 / 轻量自然流量 / 定向分享获取兴趣信号

### 3. 商业侧默认采用保守判断

- 价格内部参考带：`$109-$139`
- 最小可行履约判断：`中国直发美国 + $149`
- 美国仓 / 3PL 后置，不作为当前第一路线

## 本周最合理的动作顺序

1. 技术：完成首页 section、导航页脚、PDP 推送、draft 商品、邮件收集、GA4。
2. 内容：把 `1+2+2` 结构正式挂进首页与 notify-only PDP。
3. 商业：以 `不展示价格 + 收集通知` 的方式外放预览页。
4. 决策：观察兴趣信号后，再决定是否进入样品验证、真实报价刷新或下一阶段履约深化。

## 当前明确不做

- 不新增询盘
- 不新增供应商搜索
- 不下单
- 不支付样品费
- 不扩成多 SKU 正式上架
- 不把咖啡壶写成银内壁款
- 不公开承诺价格、库存、交期、控流或检测结果

## 什么时候应转为 No-Go

出现以下任一情况，应停止继续推进：

- 外部接受带明显低于 `$149`
- 真实跨境物流 / 批量成本高于当前保守模型，导致最小可行组合失效
- 页面最终仍无法形成足够信任，只剩“空店 + 单商品壳”

## CEO 可直接复述给董事会的结论

这条 Capsule Preview 路线可以继续，但只能按低承诺验证推进，而不是当成正式开卖。我们已经把内容结构、技术骨架和最小履约判断都补齐了：现在能做出一个可信的预览态页面，并收集到货通知。但从商业上看，低价带不成立，真正勉强可行的是 `中国直发美国 + 中档价格假设`。所以本周最合理的动作不是去卖货，而是先把预览态做完整，拿兴趣信号，再决定是否进入下一阶段。
